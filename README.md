# AI Epoch — 更自由的 Galgame

> 自由度应该是很多玩家都在追求的东西。比如你玩到某个剧情，屏幕只弹出了三个选项——「好吧」「不行」「再说」——你心里面想对角色说的话再多也无法传达到，因为游戏开发者并没有给你这个机会。

**AI Epoch 把这个机会还给你。**

不需要等开发者写剧情分支，不需要从三个预设选项里勉强选一个。你打字，AI 以角色身份实时回应。你想说什么就说什么，想去哪就去哪，想怎么发展就怎么发展。

---

## 🎮 这是什么

一个 **AI 驱动的开放叙事引擎**。前端是聊天界面，后端是大语言模型。

和传统 Galgame 的区别：

| 传统 Galgame | AI Epoch |
|:--|:--|
| 开发者写死 N 条分支 | 无限分支，AI 实时生成 |
| 三选一 / 五选一 | **你打什么字都行** |
| 只有预设剧情能触发回应 | 你说的每句话都能得到回应 |
| 角色是死的数据 | 角色会「理解」你说的话 |
| replay 看相同内容 | 每次输入都不同，怎么玩都新鲜 |

## 🚀 快速开始

```bash
pip install -r requirements.txt
python server.py
```

Web 界面：http://127.0.0.1:8766  
CLI 客户端：`python cli.py`

---

## ⚙️ 配置

点右上角 ⚙ 设置，或者编辑 `data/config.yaml`（首次启动自动生成）：

```yaml
ai:
  api_base: https://api.openai.com/v1
  api_key: sk-xxx
  model: gpt-4o-mini
port: 8766
```

兼容所有 OpenAI 格式的 API（ChatGPT / Gemini / DeepSeek / 本地模型等）。

---

## 🎭 怎么玩

打开网页，角色已经在等你了。你可以：

- **自由对话**：说什么都行，AI 会以角色身份回应
- **推进剧情**：描述你想做的任何事，AI 即时生成后续发展  
- **切换角色**：在设置里修改系统提示词，改变 AI 的人设和世界观

---

## 🏗️ 技术架构

```
浏览器/CLI → Flask Server → OpenAI 兼容 API → AI 回复
```

- **server.py** — Flask 后端，管理 AI 连接和对话会话
- **cli.py** — 终端客户端，支持彩色输出和命令交互
- **index.html** — Web 聊天界面，毛玻璃暗色 UI

---

## 🧠 为什么不做存档系统

故意的。

存档意味着可以回头重来——但真正的自由是「每一个选择都是唯一的」。没有 SL 大法，没有最优选项。你说过的话就是你说过的，AI 记住了，故事继续往前走。就像真实的人生。

（如果你确实需要清空重新开始，点「新对话」即可。）
